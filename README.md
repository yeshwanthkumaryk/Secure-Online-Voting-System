# Secure-Online-Voting-System

This project is a web-based Secure Online Voting System developed using Java, JSP, Servlets, and MySQL. The application allows users to register, log in, and cast their votes securely through a browser-based interface.

The system ensures that each voter can vote only once and maintains data integrity using database validation. It includes features such as user authentication, password reset functionality, and dynamic candidate selection.

The frontend is designed using Bootstrap to provide a responsive and user-friendly interface, including navigation bar, carousel, and interactive voting pages. The backend follows the MVC architecture using Servlets as controllers, JSP for views, and DAO classes for database operations.

This project demonstrates full-stack Java web development with secure data handling and real-world application logic.
